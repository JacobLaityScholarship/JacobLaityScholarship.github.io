Phaser

A brief review

Phaser Sandbox is an online JavaScript code editor that contains many tools to help you build 2D games in JavaScript. There is also Phaser which can be coded in a text editor such as SubLime Text. Phaser has built in sprites and physics, it is a little bit like scratch but you actually have to do JavaScript coding. I really like phaser as you can easily learn JavaScript with the many phaser tutorials and at the same time make a cool 2D game! This is a good introduction into the unforgiving world of JavaScript
