Jacob's Journey

I started out in software by creating simple games on scratch. After spending a few months on scratch I moved on to pyhton, I built a robot and a few basic minecraft plugins. Both of these languages are good for building up logic and problem solving for future coders. Next off I started building web pages with HTML. I then decorated them with CSS and made them interactive with JavaScript. These three languages work VERY well together and are perfect for building websites and webapps. Next up I started programming more complex minecraft plugins with Java. Later on I found out about Phaser - this again looping back to HTML and JavaScipt but this time building 2D games. Currently I am attempting to have a go at swift to create apps for IOS but it is pretty difficult.

Recommended Journey

We recommend for new coders to buid up logic on scratch and then move on to python. After spending some time on python we recommend looking into HTML, CSS and JavaScript to build some WebApps - bearing in mind JavaScript can be fairly confusing at first. We also recommend having a quick look at Phaser as it is a fun thing to use. After you have done all of this our recommendation is to start Swift and developing iOS apps.
