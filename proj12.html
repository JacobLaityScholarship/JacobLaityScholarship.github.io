2D iOS Games

Using Apple's Spritekit


Spritekit is officially-support by Apple and is built right into Xcode, it is perfect for 2D game development for iOS, Apple Watch and Apple TV. It’s closest competitor would likely be the Unity engine, probably the more powerful and flexible of the two - Spritkit’s main advantage is being built right into Xcode and being officially supported by Apple.

In Spritekit I designed 3 games.

3D touch game

Video of Game

Inspired by DividR

A game that the player controls using Apple’s 3D Touch (Force Touch) functionality - it allows devices to distinguish between different levels of force and respond accordingly. It is based off the Award Winning app DividR.
The player is a cube - The harder they press the screen the further the cube splits apart. The aim of the game is to allow the cube(s) to navigate through an obstacle course.

Space Game

Video of Game

Inspired by Space Invaders

A space-shooter, much like space invaders, but the user must tilt there phone to move their ship and tap the screen to fire. The aim of the game is to destroy ALL the obstacles, if one of the obstacles passes the player, they lose 1 of their 3 lives. The game is infinite, and uses a score based system.
